## 2.标题语法：

```javascript
/*
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
*/ 
```

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 3.字体语法：

```javascript
**加粗**
*斜体*
***斜体加粗***
==高亮==
~~删除线~~
```

**加粗**
*斜体*

***斜体加粗***

==高亮==
~~删除线~~

## 4.引用：

```javascript
//引用语法
>作者：嘿嘿
>>作者：嘿嘿
>>>作者：嘿嘿
```

>作者：嘿嘿
>>作者：嘿嘿
>>>作者：嘿嘿
